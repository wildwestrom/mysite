## Develop

Before running for the first time run:
`yarn install`

### CLI

`yarn watch`

### [CIDER](https://cider.mx/) (Emacs)

- `M-x`
- `sesman-start`
- Select command: `cider-connect-cljs`
- host: `localhost`
- port: `3333`
- repl-type: `shadow`
- build: `node-repl`
